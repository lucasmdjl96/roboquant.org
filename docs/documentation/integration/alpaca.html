Alpaca is an API first broker. Their focus is mainly on US stocks, but they also added cryptocurrencies.

You can find out more at their website at https://alpaca.markets/

The integration with Alpaca covers the following aspects:

  1. Get historic data

  2. Get live data

  3. Use the broker account (both paper trading and live trading)

import org.roboquant.alpaca.*

Historic Data

Alpaca provides a nice set of historic data that you can use during back testing.

val feed = AlpacaHistoricFeed()
val tf = Timeframe.past(100.days)
feed.retrieveBars("AAPL", "IBM", timeframe = tf)

Live Data

Next to historic data, Alpaca also provides live data feeds.

val feed = AlpacaLiveFeed()
feed.subscribe("AAPL", "IBM")

val tf = Timeframe.next(120.minutes)
roboquant.run(feed,tf)

Broker

val broker = AlpacaBroker()
broker.account.summary().log()
