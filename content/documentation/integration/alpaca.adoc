Alpaca is an API first broker. Their focus is mainly on US stocks, but they also added cryptocurrencies.

image::https://roboquant.org/img/thirdparty/alpaca.jpg[]

You can find out more at their website at https://alpaca.markets/

The integration with Alpaca covers the following aspects:

1. Get historic data
2. Get live data
3. Use the broker account (both paper trading and live trading)

----

----
