OANDA provides several types of CFD instruments, including indices, forex, commodities, metals and bonds. You can find more about OANDA by visiting their website at https://www.oanda.com

image::http://roboquant.org/img/thirdparty/oanda.jpeg[oanda, 500]

The integration with OANDA covers the following aspects:

1. Get historic data
2. Get live data
3. Use the broker account (both paper trading and live trading)
